{{< img-text title="Kód a architektura" src="/images/Coding.png" >}}
Tento projekt se zaměřuje na vybudování osobního portfolia od základů, přičemž jsem se záměrně vyhnul použití hotových webových editorů, abych měl plnou kontrolu nad zdrojovým kódem. Celý vývoj probíhal ve Visual Studio Code a web je poháněn frameworkem Hugo – rychlým a flexibilním generátorem statických stránek (SSG). Pro architekturu frontendu jsem využil HTML v kombinaci s Tailwind CSS, jehož "utility-first" přístup mi umožnil zefektivnit stylování a udržet kód čistý a responzivní.
{{< /img-text >}}

{{< img-text title="Filozofie designu" src="/images/Bento-grid.png" reverse="true" >}}
Vizuální identita je ukotvena v estetice tmavého režimu (dark mode), která působí moderně, elegantně a méně namáhá oči. Pro vytvoření jasné vizuální hierarchie jsem integroval vysoce kontrastní neonové prvky, které přitahují pozornost ke klíčovým informacím. Obsah je organizován pomocí Bento mřížky (Bento grid layout), která dělí informace do modulárních, interaktivních dlaždic. Pro zlepšení uživatelské zkušenosti (UX) jsem implementoval plynulý motion design – včetně hladkých přechodů a mikro-interakcí – což zajišťuje intuitivní a plynulý průchod stránkou.
{{< /img-text >}}

{{< img-text title="Nasazení" src="/images/GitHub.png" >}}
Závěrečnou fází bylo nasazení statického webu prostřednictvím GitHub Pages. Tento pracovní postup (workflow) využívá git pro důslednou správu verzí, což mi umožňuje efektivně sledovat historii vývoje a spravovat jednotlivé větve projektu. Toto nastavení výrazně urychluje iterační cyklus, umožňuje mi rychle identifikovat a ladit chyby (debug) a okamžitě publikovat nový obsah a aktualizace na živý server.
